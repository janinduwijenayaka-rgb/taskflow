# ✓ TaskFlow — Task Manager

A clean, modern Kanban-style task manager built with plain HTML, CSS, and JavaScript. No frameworks, no dependencies — just open `index.html` and it works.

## ✨ Features

- **Kanban Board** — To Do / In Progress / Done columns
- **Priority Levels** — High 🔴 / Medium 🟡 / Low 🟢
- **Categories** — Assignment, Project, Exam, Personal, General
- **Due Dates** — Overdue detection with ⚠ warning
- **Progress Bar** — Visual completion percentage
- **Persistent Storage** — `localStorage` ෙකොරෙන් data save (page refresh කළත් data stays)
- **Responsive Design** — Mobile + Desktop support
- **Keyboard Support** — Enter key ෙකොරෙන් task add

## 📖 How to Use

1. **Task add කරන්න** — Task name type කරලා priority, category, due date select කරලා **+ Add task** click කරන්න (හෝ Enter press)
2. **Task move කරන්න** — **Start →** button ෙකොරෙන් In Progress ට, **Complete ✓** ෙකොරෙන් Done ට
3. **Task delete කරන්න** — **Delete** button click
